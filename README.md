# Football Tournament Simulator

This is a simple C program that simulates a football tournament.

The tournament starts with a **group stage** (8 groups of 4 teams).  
Each team plays against the others in its group and earns points based on the result.

After the group stage, the **top two teams from each group qualify** for the knockout phase.

The knockout phase includes:
- Round of 16
- Quarter-finals
- Semi-finals
- Final

If a match ends in a draw during the knockout phase, the winner is decided with **penalty shootouts**.
